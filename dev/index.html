MobiusTransformations.MobiusTransformation — Type
MobiusTransformation{T}

Represents a Möbius transformation of the form: f(z) = (az + b) / (cz + d)

Fields

  • a::T, b::T, c::T, d::T: Coefficients of the transformation

MobiusTransformations.MobiusTransformation — Method
(m::MobiusTransformation)(z)

Apply to a complex number z the Möbius transformation m(z) = (a*z + b) / (c*z + d), where m = Mobius([a b; c d]).
